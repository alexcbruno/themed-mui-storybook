# Generate a Figma library out of Storybook components

Build a shareable Figma file out of your storybook components to make sure you're designing your UI as your users would see it.

Try it out with this sample by clicking on the Anima button on the topbar to export the full library, or select a component to export from the left sidebar (like this [**Button**](/story/material-ui-button--button) for example) and choose “Export this component”.
